# Release v2.3.0

**Date:** 2025-11-21

## Overview
This release completes the core management features of the Admin Dashboard by adding full Device Management capabilities. Administrators can now view and delete devices system-wide.

## New Features
- **Admin Device Management:**
  - View all devices across all tenants.
  - Delete devices directly from the Admin Panel.
  - View device details including IP, Public Key, and Last Seen status.

## Improvements
- **Admin Dashboard:**
  - Unified management interface for Users, Tenants, Networks, and Devices.
  - Improved data loading performance with parallel requests.

## Fixes
- Fixed an issue where device listing was restricted to the admin's tenant only.

## Installation
### Server
```bash
# Download the binary
wget https://github.com/orhaniscoding/goconnect/releases/download/v2.3.0/goconnect-server-linux-amd64
chmod +x goconnect-server-linux-amd64
./goconnect-server-linux-amd64
```

### Client
```bash
# Linux
wget https://github.com/orhaniscoding/goconnect/releases/download/v2.3.0/goconnect-client-linux-amd64
chmod +x goconnect-client-linux-amd64
sudo ./goconnect-client-linux-amd64 install

# Windows
# Download goconnect-client-windows-amd64.exe and run as Administrator
```
