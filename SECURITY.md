# Security Policy

## Supported Versions

Only the latest version of GoConnect is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within GoConnect, please send an email to security@goconnect.io. All security vulnerabilities will be promptly addressed.

### What to Include

*   Description of the vulnerability.
*   Steps to reproduce the issue.
*   Potential impact.

We appreciate your help in keeping GoConnect secure!
